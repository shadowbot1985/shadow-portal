# 超能力入口網站技能流程

學習來源：

```text
https://hanassitant-cloud.github.io/jarvistest/partner-superpower-portal-skill/#full-doc
```

技能代號：`partner-superpower-portal`

這個技能不是 HTML PPT。HTML PPT 是簡報製作能力；`partner-superpower-portal` 是用來建立每個小伙伴自己的 GitHub Pages 超能力入口網站。

## 核心用途

幫每位小伙伴建立自己的公開入口網站，集中放：

- 主入口
- 所有網址列表
- HTML PPT 目錄
- 技能列表
- Google Drive 檔案總管
- 可選的客服入口、AI 整合網站、專題分析報告

## 開工前資料表

開工前要先確認：

- 小伙伴名稱、公開名稱、角色
- 本機工作目錄
- 歸檔位置與備份資料夾
- 自己的 Gmail
- 自己的 GitHub 帳號、repo、Pages URL
- 自己的 Google Drive remote、資料夾、分享規則
- 公私資料邊界
- 最終入口網址 slug

不能亂填、猜填或照抄其他小伙伴的資料。能從本機查到的先查；不能安全確認的標記待核對。

## 標準網站功能

1. 主入口：第一頁顯示小伙伴名稱、角色和所有功能方塊。
2. 所有網址列表：整理官網、社群、客服、AI 系統、公開工具。
3. HTML PPT 目錄：公開簡報依日期由新到舊排列。
4. 技能列表：只放技能名稱、簡短用途、狀態和大分類。
5. 檔案總管：整理 Google Drive 公開檔案、檔名、說明、日期、預覽和下載連結。

## 公開程度

小伙伴入口網站可能會給客戶看，所以只秀應用層：

- 可以公開：服務能力、成果、公開作品、客戶能理解的用途
- 不公開：內部流程、腳本、部署方式、帳號設定、完整 SOP、token、密碼、後台連結、客戶私密資料
