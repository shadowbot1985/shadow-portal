### Google Drive

```sh

```

建議設定：

- remote name：`shadowdrive`
- storage：Google Drive
- scope：drive
- root folder：留空
- service account：留空
- auto config：yes

完成後測試：

```sh
rclone lsd shadowdrive:
```
