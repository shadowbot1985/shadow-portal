# HDMI PPT 標準製作流程

## 1. 需求確認

- 客戶名稱
- 主題與用途
- 螢幕比例或播放設備
- 交付格式
- 截止時間

## 2. 素材整理

- 原始素材放入 `02_素材_raw`
- 已選素材放入 `03_圖片_selected`
- 音樂放入 `04_音樂_music`
- 不確定授權的音樂不得進入正式製作

## 3. 圖片檢查

- 主體完整
- 產品完整
- 人物沒有被切掉
- 不做非等比縮放
- 手機與大螢幕都能舒服觀看

## 4. 音樂檢查

- 情緒符合內容
- 節奏支撐畫面
- 不恐怖、不陰沉、不像送葬
- 逐首確認錄音檔授權

## 5. PPT 製作

- 文字少而清楚
- 重點放在安全框內
- 圖片不變形
- 轉場服務內容，不搶戲

## 6. 輸出交付

- 輸出前跑檢查表
- 檔名包含日期與版本
- 最終檔放入 `08_交付_delivery`
