# 影子 Shadow 超能力入口網站

這是 Shadow 的固定超能力入口網站，依 `partner-superpower-portal` 標準重建，並對齊既有成功案例的 URL 配置。

## 固定入口

- GitHub Pages: https://shadowbot1985.github.io/shadow-portal/
- Repository: https://github.com/shadowbot1985/shadow-portal

## 標準功能

- `index.html`：主入口
- `websites.html`：所有網址列表
- `file-manager.html`：Google Drive 檔案總管
- `ppt-catalog/index.html`：HTML PPT 目錄
- `shadow-skills.html`：技能列表
- 公私邊界與公開規則

## 公開邊界

公開頁只放服務能力、公開作品、公開連結與已授權交付檔案。不放內部 SOP、部署細節、腳本、帳號權限、私人備份或客戶敏感資料。

## 學到的規則

不要自己創造 URL 結構。小伙伴入口網站要先模仿成功案例的檔名、相對連結和頁面配置，再替換成自己的名稱、內容與公開資料。
